攻防矩阵

在对云服务进行研究的过程中，火线安全将研究的成果进行总结归纳整理，提出了云服务攻防矩阵。

该矩阵从前期侦查、初始访问、执行、权限提升、权限维持、防御绕过、信息收集、横向移动和影响这九个维度展开，分别研究了每个维度下，云服务可能面临的威胁。

通过这套攻防矩阵，红队在面临云环境时也将有章可循，明白自己的下一步该如何去走。
